# Reaction Warrior
It's a game built with C++ that works in your terminal that tests your reaction speed when you see a letter and a number pop up on your screen. (work in progress)

## Features

Difficulty Modes: Easy, Medium, and Hard
Tracks how long you took to type the time 
Records your score
Ranking system

## Stuff used for this 

- C++

# HOW TO RUN
- Download the "main.exe" file.
- Select a difficulty.
- Try to enter each phrase as quickly as you can.
